## Project Description
Wordly is a simple Single Page Application (SPA) dictionary.  
It lets users search for English words and see the definition, part of speech, examples, synonyms, and a link to read more.  

The app uses JavaScript to fetch data from an online dictionary API and shows the information without reloading the page.


## Features
- Search for any English word.  
- Shows word meaning, part of speech, example sentence, and synonyms.  
- Read More button links to the source website.  
- Error message appears if the word is not found.  
- Works on desktop and mobile screens.  


## Technologies Used
- HTML  
- CSS  
- JavaScript  
- [Free Dictionary API](https://dictionaryapi.dev/)


## Setup Instructions
1. Clone the repository:  
git clone <https://github.com/Mwihaki112/SPA-wordly.git>
2. Navigate to the project folder
3. Open index.html in a web browser, it runs entirely in the browser.

## How to use
1. Type a word into the search bar.
2. Click the Search button.
3. The app will show:
   Word
   Part of speech
   Meaning
   Example sentence
   Synonyms
   Read More link
4. If the word is not in the dictionary, the app will display: Sorry, word not found
